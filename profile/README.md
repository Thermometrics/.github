# Thermometrics Corporation

Thermometrics Corporation is a U.S. manufacturer of custom temperature sensors, RTDs, thermocouples, bearing sensors, thermowells, fittings, calibration support, and temperature measurement hardware for demanding industrial and aerospace applications.

Since 1965, Thermometrics has supported engineers, buyers, maintenance teams, and quality groups that need reliable thermal measurement assemblies built around real installation requirements instead of catalog-only sensor options.

## Custom Temperature Sensor Manufacturing

Thermometrics designs and manufactures temperature sensor assemblies for applications where fit, response time, temperature range, mounting method, material compatibility, documentation, and reliability all matter.

Core capabilities include:

- Custom thermocouple assemblies
- Pt100 and Pt1000 RTD temperature sensors
- Bearing RTDs and bearing thermocouples
- Intrinsically safe bearing sensor assemblies
- Mineral insulated metal sheath thermocouples
- Industrial head temperature sensors
- Multipoint thermocouples and RTDs
- Surface mount and clamp-on RTDs
- Sanitary RTD assemblies
- High-pressure and severe-duty thermocouples
- Thermowell-mounted RTDs and thermocouples
- Machined thermowells, fittings, adapters, and sensor hardware
- Calibration support and documentation for temperature sensor programs

## Industries Served

Thermometrics supports temperature measurement programs across aerospace, defense, industrial machinery, energy, power generation, oil and gas, refining, petrochemical, heat treating, automotive, EV manufacturing, HVAC, refrigeration, plastics, medical manufacturing, semiconductor, electronics, research, development, water, environmental, food, beverage, pharmaceutical, and biotech applications.

Buyers and engineers often work with Thermometrics when a standard probe does not solve the full problem. Common requirements include custom sheath materials, special fittings, defined lead exits, bearing sensor tip geometry, calibration points, material traceability, dimensional inspection, and documentation aligned to customer quality systems.

## RTDs, Thermocouples, and Thermowells

Thermometrics manufactures platinum RTDs, including Pt100 and Pt1000 sensors, along with thermocouple assemblies for Type K, Type J, Type T, Type E, Type N, Type R, Type S, and Type B applications.

The company also supports thermowell and fitting requirements for process instrumentation, industrial equipment, aerospace programs, and engineered assemblies where the sensor, connection, protection, and documentation need to work together.

## Bearing Temperature Sensors

Thermometrics manufactures bearing temperature sensors for rotating equipment, machinery protection, and industrial monitoring applications. Bearing RTDs and bearing thermocouples can be configured around case style, lead protection, installation geometry, and environmental requirements.

Hazardous-area and intrinsically safe bearing sensor programs may involve ATEX, IECEx, UKEX, NRTL, CSA, Zone 0/1/2, and Class I, Division 1 requirements depending on the product family and application.

## Quality, Traceability, and Calibration Support

Thermometrics supports programs where quality documentation is part of the purchase decision. Available support may include material traceability, dimensional inspection, calibration documentation, sensor identification, and application-specific review.

Relevant quality and documentation signals include:

- AS9100D quality system support
- ISO 9001:2015 quality system support
- NIST-traceable calibration documentation where applicable
- IEC 60751 RTD tolerance context
- Thermocouple reference standards and color code guidance
- Application support for custom sensor assemblies and replacement builds
